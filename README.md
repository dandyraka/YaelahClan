# YaelahClan
check availability of social media usernames for Yaelah Clan (instagram &amp; twitter)

## Install
Download [Composer](https://getcomposer.org/download/).

```bash
composer install
```

## Run
```bash
cd YaelahClan/src
```

```bash
php run.php
```
